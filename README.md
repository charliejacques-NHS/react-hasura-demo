## Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/charliejacques-NHS/react-hasura-demo.git
   ```

2. Navigate to the project directory:

   ```bash
   cd react-hasura-demo
   ```

3. Install dependencies:

   ```bash
   npm install
   # or
   yarn
   ```
